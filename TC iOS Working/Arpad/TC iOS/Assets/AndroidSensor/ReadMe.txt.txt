Android Sensor for Unity3d

Features

1. Pedometer(step counter like fitbit but for steps only)
2. Share Detector

Note: You need actual Physical Android Device to test this Plugin

Also: Pedometer only work on Android 4.4(Kitkat) API 20


================================================================================================================================
                                                How to use
================================================================================================================================

1. Download the plugin from unity3d asset store 
https://www.assetstore.unity3d.com/en/#!/search/page=1/sortby=popularity/query=publisher:13227
https://www.assetstore.unity3d.com/#!/content/43522

2. Check and import all the files from plugin

3. Inside Unity3d go to File Menu, Build Settings and then select player settings and look for other settings
and change the bundle indentifier to your game or app bundle identifier for ex. com.mycoolstudio.awesomegame

4. On Build Settings switch platform to android platform

5. Inside Unity3d in your Project hierarchy go to "Assets/AndroidSensor/Scenes" and open the Demo scene

6. Inside Unity3d go to File Menu, Build Settings and then make sure that you added Demo Scene

7. Inside Unity3d go to File Menu, Build Settings Switch Platform to Android

8. Inside Unity3d go to File Menu, Build Settings and then select Build and Run

9. wait for the build and then check the application if everything is working fine and that's it.
